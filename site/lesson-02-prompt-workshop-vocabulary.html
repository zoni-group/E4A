4.1 Start Here

In Lesson 1 you wrote your first safe English AI prompt and checked privacy settings. Today you will make that prompt stronger. You will try small changes, compare the answers, and build your vocabulary log.

You can use an AI tool, a partner, or teacher-provided sample outputs. You do not need AI access to complete the portfolio artifact.

4.2 Why This Matters

A small change in your English can make a big change in the AI answer. When you add context, limits, and a clear format, the output is easier to read and check. This lesson helps you practice prompt writing as an English skill.

4.3 Today You Will

By the end of this lesson, you will:

  1. Review the ZONI CLEAR prompt scaffold.
  2. Write three prompt variations for one safe task.
  3. Compare AI outputs or sample outputs.
  4. Add useful words to your vocabulary log.
  5. Save: English AI Decision Toolkit/Lesson 2 - Prompt Practice and Vocabulary Log.md.
NoteLesson Reference

Theme: Clear prompt changes improve AI answers and English practice.

AI Focus: Prompt variations and output comparison.

ESL Focus: Request language, imperatives, vocabulary logging, and reason-giving.

Content Objective: Students will compare three prompt versions and choose the strongest one.

Language Objective: Students will explain prompt choices using simple English:

Prompt ___ was clearer because...
I added context about...
The answer was useful because...
I should verify...

4.4 Core Idea

A prompt variation is a different version of the same prompt. You might make one prompt shorter, one prompt with more context, and one prompt with a clear format.

Comparing outputs helps you ask:

Which answer is clearer?
Which answer is safer?
Which answer can I check?
Which answer uses English I understand?

4.5 Key Vocabulary

Word Simple Meaning Example Sentence
prompt The instruction or question you give AI My prompt asks for a study plan.
variation A different version I wrote three prompt variations.
output The answer from AI The output was easy to read.
format The shape of the answer I asked for a table format.
context Background information I gave context about my schedule.
limit A rule for the answer My limit says use simple English.
compare Look at two or more things I compare the outputs.
vocabulary log A list of new words and sentences I added five words to my vocabulary log.

4.8 Guided Practice

4.8.1 Three Prompt Variations

Task:

Make a simple vocabulary study plan.

Prompt 1 - short:

Help me study vocabulary.

Prompt 2 - more context:

Help me study 10 English vocabulary words for 20 minutes.
I am a beginner English student.
Use simple English.

Prompt 3 - clear format:

Help me study 10 English vocabulary words for 20 minutes.

Context:
- I am a beginner English student.
- I want speaking and writing practice.

Limits:
- Use simple English.
- Do not add private information.

Give me:
1. A 5-minute warm-up.
2. A matching activity.
3. Five practice sentences.
4. A short review question.

Discuss:

Which prompt is easiest to understand?
Which prompt gives the clearest output?
Which prompt is safest?
