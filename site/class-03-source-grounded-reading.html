7.1 Start Here

A friend asks you: “Can I return this item to the store?”

You see two answers.

Answer A:

Yes, you can return it within 30 days.

Answer B:

According to the store's return policy, items can be returned
within 30 days with a receipt.

Which answer do you trust more?

Answer B shows where the information came from. You can check it. Answer A might be true — but it might be wrong, and you have no way to know. Trusting an answer without a source can cost you time, money, or even a job.

7.2 Why This Matters

AI sometimes invents answers that sound correct but are not true. We call this a hallucination. If you ask AI about school rules, store policies, public notices, work guides, or health information, a wrong answer can cause real problems. Today you will learn how to give AI a short source text and ask for answers only from that source. This is a safer way to use AI, and it is excellent English reading practice.

7.3 Today You Will

By the end of this lesson, you will:

  1. Learn how to write a source-grounded prompt.
  2. Quote a short public text and paraphrase it in your own simple English.
  3. Practice saying “I cannot answer from the provided source” when the answer is not there.
  4. Spot a hallucination by checking the source.

7.4 Beginner Explanation

Sometimes AI gives an answer that sounds correct but is not really supported. We call this a hallucination. The words look fine, but the facts may be invented.

A safer way to use AI for reading: give it a short source text and ask it to answer only from that text. If the answer is not in the text, AI should say so — and so should you.

In simple English:

Do not guess. Use the source.
NoteResearch Note

This classroom routine is informed by work on AI factuality, hallucination, and retrieval/source-grounded generation. In this class, we practice a simple version: use the provided source, quote short evidence, and do not guess when the source does not answer (Maynez et al. 2020; Lewis et al. 2020).

7.5 Key Vocabulary

Word Simple Meaning Example Sentence
source The original text or information The source is a school policy.
citation A note that shows where information came from I need a citation for this answer.
evidence Words from the source that support an answer The sentence is evidence.
quote Exact words from a source I used a short quote.
paraphrase Same idea in your own words I paraphrased the rule.
hallucination An AI answer that may be wrong or unsupported The AI hallucinated a date.

7.8 Guided Practice

7.8.1 Warm-Up: Which Answer Is Better?

Answer A:

You can return the item after 30 days.

Answer B:

According to the return policy, items can be returned within 30 days with a receipt.

Discuss:

Which answer is easier to trust?
What evidence do you see?
What is still missing?

7.8.2 Mini-Lesson: Source-Grounded Prompt

Use this template:

Use ONLY the source text below.
If the answer is not in the source, say:
"I cannot answer from the provided source."

Question:
[student question]

Source text:
[paste short source section]

Answer format:
1. Short answer
2. Evidence from the source
3. What is unclear
4. What I should check next

In simple English: use the source. If the source does not answer, say so.

NoteSource-Checking Note

For important answers, stop, check the source, look for better coverage, and trace the claim back to evidence (Caulfield 2019).

7.9 Independent Practice

Ask one supported question and one unsupported question about the same source.

Supported question example:

How many days can I borrow a library book?

Unsupported question example:

What is the library manager's personal phone number?

Correct unsupported answer:

I cannot answer from the provided source.

Label each answer:

Supported / Not supported / Not in the source
